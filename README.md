# Sign In Page UI

This is a simple and modern **Login / Sign-In page UI** built using **HTML and CSS**.  
It includes a responsive layout with a clean design, email/password form, and social login buttons.

---

##  Features

- Clean and modern UI design
- Email and password input fields
- "Remember me" checkbox
- Forgot password option
- Google and Apple sign-in buttons
- Responsive layout with side image
- Simple and beginner-friendly code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
